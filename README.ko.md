# 스폰지밥 캐릭터 얼굴 매칭기

## 프로젝트 개요

여덟 가지의 *네모바지 스폰지밥* 캐릭터 중 누가 업로드된 얼굴 사진과 가장 유사한지를 예측하는 브라우저 기반 웹 프로젝트입니다.

이미지 분류 모델은 [Google Teachable Machine](https://teachablemachine.withgoogle.com/)을 사용하여 학습되었습니다. 웹사이트는 호스팅된 URL을 통해 학습된 모델을 불러와 브라우저에서 직접 예측을 수행합니다.

이 프로젝트는 2023년, 고등학교 2학년 시기, 고등학교 동아리 **Facio**의 일환으로 개발되었습니다. 이는 이미지 분류와 브라우저 기반 머신러닝의 입문용 어플리케이션으로 제작되었습니다.

프로젝트를 통해 다음 분야의 기초적인 경험을 얻었습니다.

* 사진 데이터 수집 및 정리
* 다중 클래스 이미지 범주 정의
* 이미지 분류 모델 학습
* Teachable Machine 모델 게시
* JavaScript를 활용한 업로드 파일 처리
* 브라우저 기반 모델 추론 통합
* 웹 인터페이스를 통한 분류 결과 표시

## 작동 방식

1. 사용자가 업로드 영역을 클릭하거나 사진을 드래그 앤 드롭하여 얼굴 사진을 업로드합니다.
2. JavaScript가 선택된 파일을 읽고 미리보기를 표시합니다.
3. 사용자가 **캐릭터 확인하기** 버튼을 클릭합니다.
4. 웹사이트가 Google Teachable Machine에서 이미지 분류 모델을 불러옵니다.
5. 모델이 여덟 가지의 캐릭터 클래스에 대한 예측 점수를 계산합니다.
6. 웹사이트가 가장 높은 점수의 클래스를 선택합니다.
7. 해당 캐릭터 이미지가 결과로 표시됩니다.
8. 사용자는 **이미지 제거** 버튼을 클릭하여 업로드한 이미지를 삭제하고 인터페이스를 초기화할 수 있습니다.

결과는 오직 원본 학습 이미지에서 학습된 시각적 패턴만을 반영합니다. 얼굴의 유사성에 대한 객관적인 측정이 아닙니다.

## 캐릭터 클래스

| 모델 레이블  | 캐릭터           |
| ----------- | ---------------- |
| `spongebob` | 네모바지 스폰지밥 |
| `ddung`     | 불가사리 뚱이     |
| `squid`     | 징징이           |
| `daram`     | 다람이           |
| `plankton`  | 플랑크톤         |
| `puff`      | 퍼프 선생님      |
| `Larry`     | 바닷가재 래리    |
| `pingping`  | 핑핑이           |

`Larry`의 대문자 표기를 포함하여 학습 프로젝트에서의 원본 클래스 레이블을 유지했습니다.

## 주요 기능

* 웹 브라우저를 통한 사진 업로드
* 드래그 앤 드롭 방식의 사진 입력
* 업로드된 사진 미리보기
* 여덟 가지 캐릭터 클래스에 대한 분류
* 가장 높은 예측 점수의 클래스 선택
* 캐릭터 결과 사진 표시
* 이미지 제거 및 인터페이스 초기화
* 브라우저 기반 머신러닝 추론
* 백엔드 서버 및 데이터베이스 불필요

## 사용 기술

* HTML
* CSS
* JavaScript
* Google Teachable Machine
* TensorFlow.js
* Teachable Machine Image 라이브러리

TensorFlow.js는 Teachable Machine Image 라이브러리를 통해 사용되었습니다.

## 모델

웹사이트는 다음 Teachable Machine 모델로부터 학습된 모델을 불러옵니다.

```text
https://teachablemachine.withgoogle.com/models/MVRc5pxd6/
```

모델 파일은 이 저장소에 저장되어 있지 않습니다. 모델과 외부 JavaScript 라이브러리를 불러오기 위해 인터넷 연결이 요구됩니다.

## 저장소 구조

```text
facio-web-project/
├── index.html
├── style.css
└── images/
    ├── title.png
    ├── spongebob.png
    ├── ddung.png
    ├── daram.png
    ├── plankton.png
    ├── pingping.png
    ├── squid.png
    ├── puff.png
    └── larry.png
```

### 파일 역할

| 파일 또는 디렉터리         | 역할                                           |
| ------------------ | -------------------------------------------- |
| `index.html`       | 인터페이스, 사진 업로드 논리, 모델 불러오기, 예측 및 결과 할당을 정의합니다. |
| `style.css`        | 페이지 레이아웃과 시각적 스타일을 정의합니다.                    |
| `images/title.png` | 예측 전과 초기화 후에 표시되는 기본 사진입니다.                 |
| `images/*.png`     | 예측된 클래스에 따라 표시되는 캐릭터 사진입니다.                 |

`images/`에 있는 파일들은 인터페이스용 사진 에셋입니다. 모델의 학습 데이터셋에 포함되지 않으며 모델의 입력으로 사용되지 않습니다.

## 프로젝트 실행

### 방법 1: HTML 파일 직접 열기

1. 저장소를 다운로드하거나 복제합니다.
2. 웹 브라우저에서 `index.html`을 엽니다.
3. 웹사이트를 사용하는 동안 인터넷 연결을 유지합니다.

### 방법 2: 로컬 웹 서버 실행

저장소를 복제합니다.

```bash
git clone https://github.com/Seasoning3/facio-web-project.git
cd facio-web-project
```

Python으로 로컬 서버를 실행합니다.

```bash
python -m http.server 8000
```

다음 주소를 엽니다.

```text
http://localhost:8000/
```

npm 설치, 패키지 관리자 또는 별도의 머신러닝 환경은 요구되지 않습니다.

## 개인정보 보호

선택한 이미지는 브라우저의 `FileReader` API를 통해 로컬에서 읽히며 클라이언트 측 JavaScript를 통해 모델로 전달됩니다.

이 프로젝트에는 다음을 포함하지 않습니다.

* 백엔드 애플리케이션
* 데이터베이스
* 서버 측 사진 업로드 기능
* 사진 저장 시스템

소스 코드는 의도적으로 업로드된 사진을 저장하지 않습니다. 그럼에도 사진은 주체의 동의 하에서만 사용되어야 합니다.

## 한계

* 예측 결과는 원본 학습 사진의 수량, 다양성 및 품질에 따라 달라집니다.
* 조명, 배경, 자세, 주요 색상 및 사진 구도가 결과에 영향을 줄 수 있습니다.
* 같은 사람이 사진에 따라 다른 예측 결과가 나올 수 있습니다.
* 예측 점수는 모델에 포함된 오직 여덟 가지의 클래스 사이에 대한 상대적인 점수입니다.
* 웹사이트는 가장 높은 점수의 캐릭터 사진만 표시합니다.
* 원본 학습 데이터셋은 저장소에 포함되어 있지 않습니다.
* 호스팅된 Teachable Machine 모델이 공개적으로 접근 가능한 상태로 유지되어야 합니다.
* 모델 추론에 인터넷 연결이 요구됩니다.
* 모델은 과학적인 방식으로 검증되지 않았습니다.
* 이 프로젝트는 얼굴 인식이나 심리 평가 시스템이 아닙니다.

## 면책 조항

이 프로젝트는 교육 및 오락 목적으로만 제작되었습니다. 결과에는 과학적·심리적·신원과 관련된 의미가 없습니다.

*네모바지 스폰지밥*과 그 캐릭터의 저작권 및 상표권은 각각의 권리자에게 있습니다. 이 프로젝트는 독립적으로 제작된 학생 프로젝트이며 니켈로디언, 파라마운트 또는 *네모바지 스폰지밥*의 공식 권리자와 제휴하거나 이들의 승인 또는 후원을 받지 않았습니다.
